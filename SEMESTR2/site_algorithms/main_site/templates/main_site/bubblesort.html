Bubble Sort The Simplest Sorting Algorithm

Bubble sorting is a sorting algorithm that runs through the list several times. On each pass, two neighboring items are compared, and if they are in the wrong order, they are swapped. The passes are repeated until all the items in the list have been sorted from left to right.

Bubble sorting algorithm:

  1. Pass from the 0th element to n-1 elements of the list.
  2. For each i-th element inside this loop, check if ai > ai+1, then swap ai and ai+1. If the inner loop didn't change anything, we finish the algorithm.

Example code for bubble sorting in Python:

Bubble Sort Code Example 

def bubble_sort(arr):
    """
    Bubble sort implementation.
    
    :param arr: list of elements to sort
    :return: the sorted list of elements
    """
    n = len(arr)
    # Cycle through all the elements in the list
    for i in range(n):
        # the last (n-1-i) elements are already sorted,
        # since the largest element is shifted to the end of the list
        for j in range(n - i - 1):
            # We compare the two adjacent elements
            if arr[j] > arr[j + 1]:
                # if the first element is larger than the second,
                # swap them in places
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

The above bubble_sort() function takes a list of elements arr. The function first determines the length of the list n, then the outer for loop goes through all the elements of the list.

The inner for loop goes through each list item and compares it to its right neighbor. If it is larger than its right neighbor, they swap places. After the inner loop is passed, the comparisons and moves continue until all of the list items are in the correct order. The function returns a sorted list of elements.
