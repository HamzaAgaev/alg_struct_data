Insertion Sort Insert and Offset

Insertion sorting is a sorting algorithm that sorts an array of elements by alternately inserting each element in its place in the sorted part of the array. The algorithm runs through all elements of the array, starting from the second element, and each time inserts the current element into its place in the sorted part of the array.

Insertion sorting is great for small arrays or for arrays where most of the elements have already been sorted. In general, the time to perform inset sorting is O(n^2), where n is the number of elements in the array.

The advantage of sorting with inserts is that the sorting algorithm runs through all the elements only once, and the amount of memory needed to perform the sorting is proportional only to the size of the array.

The algorithm for sorting with inserts is as follows:

  1. Locate at the beginning of the unsorted part of the array.
  2. Take the first element of the unsorted part and insert it at the desired location in the unsorted part.
  3. Shift all elements from the specified location to the end of the sorted part one element to the right.
  4. Repeat steps 2-3 for all remaining elements in the unsorted part of the array.

An example of Python implementation of an insertion sorting algorithm:

Insertion Sort Code Example 

def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr

The insertion_sort function takes an array arr as input and sorts it using the insertion sort method, returning a sorted array.

At each iteration of the for loop, the current element of the array key is taken and inserted at the right place in the sorted part of the array, using the while loop. Within the while loop, the elements of the sorted part are shifted one element to the right and each time the new element is compared with the next element of the sorted part to determine the correct place of insertion.
