Merge Sort Divide and Rule

Merge Sort is a classic sorting algorithm that works on the basis of divide and conquer principles.

The basic principle of merge sorting is to divide an unsorted array into two parts, sort each part using a recursive call to merge sort and then merge the sorted subarrays.

The merge sort procedure consists of the following steps:

  1. Splitting the array: the initial array is divided into two equal parts (if the length of the array is even) or into two parts with a small deviation from equality (if the length of the array is odd).
  2. Recursive sorting of the halves: Each of the array halves is recursively sorted using the same procedure.
  3. Merge sorted parts: after ever two halves of the array are sorted, they are merged into a new array which is also sorted.

The algorithm for merging two sorted arrays is as follows:

  1. Using two pointers, we move through each array, comparing the values of the current elements.
  2. The smaller element of the two is added to the resulting array and the pointer in the array from which it was selected is moved.
  3. The process is repeated until one of the arrays is completely moved to the resulting array.
  4. The remaining elements of the other array are added to the end of the resulting array.

Here is an example of Python code that implements merge sorting:

Merge Sort Code Example 

def merge_sort(array):
    If len(array) < 2:
        return array
    else:
        mid = len(array) // 2
        left = merge_sort(array[:mid])
        right = merge_sort(array[mid:])
        return merge(left, right)

def merge(left, right):
    result = []
    i = 0
    j = 0
    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result += left[i:]
    result += right[j:]
    return result

In this example, the merge_sort function is responsible for splitting the array into two halves and sorting them recursively, and the merge function merges the two sorted arrays (left and right) into one sorted array.
