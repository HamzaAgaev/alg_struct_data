Selection Sort Find the minimum and move it up

The Selection Sort Algorithm is a simple sorting algorithm that searches for the minimum element in the array and moves it to the first place. Then the procedure is repeated for the subarray, excluding already sorted elements, and so on until all elements have been sorted.

The Selection Sorting Algorithm consists of the following steps:

  1. Find the minimum element in the array.
  2. Swap the found minimum element with the first element in the array. Exclude the first element from consideration and continue this process until all elements are sorted.

An example implementation of sorting by selection in Python:

Selection Sort Code Example 

def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        # Find the index of the minimum element
        min_index = i
        for j in range(i+1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        # change the current element with the minimum
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

The selection_sort() function takes an array arr as input and returns its sorted instance. The algorithm consists of two nested loops. The outer loop is executed n times, where n is the length of the array, and at each iteration selects the minimum element in the subarray, starting from the current position i. The inner loop goes through the remaining elements and updates the index of the minimum element if it finds an element that is smaller than the current minimum element. The current element then swaps with the minimum element found in the current iteration.
